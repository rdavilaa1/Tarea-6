## Descripción del Proyecto

Este proyecto consiste en la implementación de un Árbol B configurable por grado utilizando el lenguaje de programación Python. El sistema permite realizar operaciones fundamentales sobre la estructura de datos, así como cargar información desde archivos CSV y generar una representación gráfica del árbol utilizando Graphviz.

---

# Funcionalidades

El programa permite realizar las siguientes operaciones:

- Configurar el grado del Árbol B.
- Insertar claves.
- Buscar claves.
- Eliminar claves.
- Mostrar recorrido del árbol.
- Cargar datos masivos desde archivos CSV.
- Generar representación gráfica del Árbol B en formato PNG.
